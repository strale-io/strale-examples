# Strale + LangChain Examples

Install: `pip install langchain-strale langchain-openai`

Use all 250+ Strale capabilities as LangChain tools in your agents.

## Examples

| File | What it shows |
|------|---------------|
| `basic_tool.py` | Call a single capability directly — no agent, no LLM needed |
| `agent_with_strale.py` | Full agent that picks the right Strale tool for your request |
