# CrewAI + Strale Examples

These examples show how to use [crewai-strale](https://pypi.org/project/crewai-strale/) to give CrewAI agents access to 250+ data capabilities.

## Install

```bash
pip install crewai-strale
```

## Examples

| File | What it shows |
|------|---------------|
| `basic_tool.py` | Call a single Strale capability directly — no agent needed |
| `agent_with_strale.py` | Full CrewAI agent with access to all Strale tools |

## Get an API key

Sign up at [strale.dev](https://strale.dev) — new accounts get free trial credits.
