# Strale curl Examples

No SDK needed. Just curl.

5 capabilities are free — no API key required: `email-validate`, `dns-lookup`, `json-repair`, `url-to-markdown`, `iban-validate`.

## Examples

| File | Description |
|------|-------------|
| `free-email-validate.sh` | Validate email (free, no API key) |
| `free-dns-lookup.sh` | DNS lookup (free, no API key) |
| `paid-company-lookup.sh` | Company data lookup (requires API key) |
| `dry-run.sh` | Preview cost before executing |
| `list-capabilities.sh` | Browse available capabilities |
